# PRD：线下展 MMD / Vocaloid 工程寄存与匹配码下载

## 1. 背景
在线下展摊位中，用户会产出 MMD/Vocaloid 工程文件与导出成品。现场需要一个低门槛网页：上传后立即生成匹配码，后续可凭码再次下载。

## 2. 目标
- 用户 2 分钟内完成上传
- 生成唯一匹配码（6-8 位）
- 凭匹配码找回并下载自己的文件
- 默认 14 天自动过期清理

## 3. 用户流程
1. 进入页面 → 选择上传
2. 上传工程/导出文件（支持多文件）
3. 系统返回匹配码 + 下载入口
4. 用户离场后输入匹配码下载

## 4. 功能需求
### 4.1 上传
- 多文件上传
- 上传进度显示
- 文件大小限制（可配置）
- 上传成功后生成唯一匹配码

### 4.2 下载
- 输入匹配码查询
- 展示文件列表
- 支持逐个下载 / 打包下载（打包可后续）

### 4.3 记录与生命周期
- 保存上传时间、过期时间、下载次数
- 过期自动变更为 expired

### 4.4 管理端（后续）
- 码查询
- 失效/删除
- 记录导出

## 5. 数据模型（MVP）
- UploadRecord：matchCode、createdAt、expiresAt、status、downloadCount、metadata
- FileItem：fileName、size、mimeType、objectKey、checksum

## 6. 非功能需求
- 查询响应 < 1s（不含下载）
- 基础限流防爆破
- HTTPS 与对象存储加密

## 7. 验收标准
- 上传后稳定返回唯一匹配码
- 正确匹配码可下载文件
- 错误/过期码有明确提示
- 过期文件不可下载
